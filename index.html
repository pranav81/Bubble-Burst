BUBBLE-BURST

INSTRUCTIONS

The objective of this game is to remove all the bubbles from the screen by clicking on them.

You lose if a large portion of the screen is covered by bubbles.

A DANGER sign indicates that a high percentage of the screen is filled and you need to speed up.

Use the 'Pause' icon to pause/resume the game.

Press R to restart at any time.

Click on the START button to get going!
